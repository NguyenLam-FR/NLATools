Règles de nommage pour les plans d'action

  • %[MJ SUSPENDU]%
    MJ SUSPENDU
    C'est un faux positif sur le dashboard qui provient d'un bug
  • %[MJ NON CORRECTION JUSTIFIEE]% OU %[INTENTIONNEL]%
    INTENTIONNEL
    C'est un faux positif ou une erreur identifiée qui ne sera pas corrigée
  • not like %[MJ %]% and not like %MOE MJ%
    PRESTATAIRE
    PA à corriger par le prestataire
  • %[MAJ %]%and vap.action_def like ''%MOE MJ%'
    MOE MJ
    PA à corriger par le prestataire

Nommage pour les dumps

  • YYYYMMDD - ETAT - ‹numero› - ‹versionApp› - ‹ENV› - ‹VersionAIP› - ‹Commentaire›
Numero: correspond À l'état du snapshot sauvegardé peut prendre les valeurs: 1, 2 ou 3 uniquement
VersionApp: correspond au numéro de version tel que mentioné dans le git
ENV: environnement de travail peut prendre les valeurs dev ou prod
Version: correspond à la version
Commentaire: champ libre

analyseur PHP

penser à vider le répertoire C:\Users\Administrateur\.pdepend après analyse php.
emplacement du parametre pdepend si on veut le changer
1. Go to the installation directory where you have installed PHP. In my case its in C:\php
2. Go to the folder C:\[installation folder of PHP]\pear\PHP\Depend\Util\Configuration - In my case its C:\php\pear\PHP\Depend\Util\Configuration
3. Open the file Factory.php in edit mode.
4. Change the line number 107 -
$this->default->cache->location = $home . '/.pdepend';
as follows $this->default->cache->location = 'Q:/Q/DMT/.pdepend [The location of your choice]';
